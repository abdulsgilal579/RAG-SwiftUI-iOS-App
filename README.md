# 📚 Swift RAG App

A Retrieval-Augmented Generation app built natively in **Swift** that reads your documents and answers your questions — with **citations** straight from the source.

---

## ✨ Features

- 📄 Ingests your personal documents (PDFs, notes, and more)
- 🔍 Retrieves the most relevant passages for any query
- 💬 Generates accurate answers grounded in your own data
- 🔗 Surfaces inline **citations** so you can verify every claim
- 📱 Clean, native iOS experience

## 🛠️ Built With

- **Swift** & **SwiftUI** — native iOS UI
- **RAG pipeline** — embeddings + vector retrieval + LLM generation
- **Citation engine** — every answer points back to its source

---

## 🚀 Why I Built This

I wanted a personal knowledge assistant that runs natively on iOS, respects my documents as the source of truth, and never hallucinates without showing me where the answer came from.
